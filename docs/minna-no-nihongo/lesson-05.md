# Minna no Nihongo Lesson 05 (第5課)

**Status**: 🟩 Mastered
**Date Started**: 2026-07-23
**Focus**: Movement & Transport (`行きます/来ます/帰ります`), Direction Particle (`へ`), Vehicle Particle (`で`), Accompaniment (`と`), Calendar Dates (`〜日`)

---

## 📌 Overview & Objectives (ရည်ရွယ်ချက်)
- သွားတယ်/လာတယ်/ပြန်တယ် ကြိယာများကို Direction Particle `へ` သုံး၍ ပြောဆိုနိုင်ရန်။
- သွားလာသည့် ယာဉ်ကျေးလိုင်းကို Vehicle Particle `で` (ဖြင့်/နဲ့) သုံး၍ ပြောဆိုနိုင်ရန်။
- အတူသွားသည့် သူငယ်ချင်း/မိသားစုကို `と` (နှင့်/နဲ့) သုံး၍ ပြောဆိုနိုင်ရန်။
- နေ့ရက်များ (၁ ရက်နေ့ မှ ၃၁ ရက်နေ့ထိ) နှင့် လများ (`〜月`) ကို ပြောဆိုနိုင်ရန်။

## 📐 Key Grammar Patterns (သッドါ - 文法)

1. **[[grammar/particle-he|Place へ 行きます / 来ます / 帰ります]]** — *[နေရာ] သို့ သွားတယ် / လာတယ် / ပြန်တယ်*
   - Example (ひらがな): わたしは きょうとに いきます。 (ကျွန်တော်က ကျိုတိုသို့ သွားပါတယ်။)
2. **[[grammar/particle-de|Means で 行きます]]** — *[ယာဉ်/နည်းလမ်း] ဖြင့် သွားတယ်*
   - Example (ひらがな): でんしゃで いきます。 (လျှပ်စစ်ရထားဖြင့် သွားပါတယ်။)
3. **Person と 行きます** — *[လူ] နှင့်/နဲ့ သွားတယ်*
   - Example (ひらがな): ともだちと いきます。 (သူငယ်ချင်းနဲ့ သွားပါတယ်။)
