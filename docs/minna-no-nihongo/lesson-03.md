# Minna no Nihongo Lesson 03 (第3課)

**Status**: 🟩 Mastered
**Date Started**: 2026-07-23
**Focus**: Locations & Directions (`ここ/そこ/あそこ`), Prices (`いくら`), Offices & Floor counters (`〜階`)

---

## 📌 Overview & Objectives (ရည်ရွယ်ချက်)
- နေရာများ (ဒီမှာ၊ အဲဒီမှာ၊ ဟိုမှာ) နှင့် အရပ်မျက်နှာများကို ပြောဆိုနိုင်ရန်။
- နေရာ သို့မဟုတ် ဌာန ၏ တည်နေရာကို မေးမြန်း ပြောဆိုနိုင်ရန် (`〜は どこ/どちら ですか`)。
- စျေးနှုန်း မေးမြန်းရန် (`いくらですか`) နှင့် ဓါတ်လှေကား/ထပ် ရေတွက်ရန် (`〜かい/がい`)。

## 📐 Key Grammar Patterns (သဒ္ဒါ - 文法)

1. **[[vocabulary/places|Noun1 は (わ) 場所 (Place) です]]** — *Noun1 က [နေရာ] မှာ ရှိပါတယ်။*
   - Example (ひらがな): おてあらいは あそこです。 (သန့်စင်ခန်းက ဟိုမှာ ဖြစ်ပါတယ်။)
2. **Noun は (わ) どこ / どちら ですか** — *Noun က ဘယ်မှာလဲ / ဘယ်ဘက်မှာပါလဲ။*
   - Example (ひらがな): えれべーたーは どちらですか。 (ဓာတ်လှေကားက ဘယ်ဘက်မှာပါလဲ။)
3. **Noun1 の Noun2** — *Noun1 ထုတ် / Noun1 နိုင်ငံလုပ် Noun2*
   - Example (ひらがな): これは にほんの じどうしゃです。 (ဒါက ဂျပန်နိုင်ငံထုတ် ကား ဖြစ်ပါတယ်။)
4. **Noun は (わ) いくらですか** — *Noun က စျေးဘယ်လောက်လဲ။*
   - Example (ひらがな): この かばんは いくらですか。 (ဒီအိတ်က စျေးဘယ်လောက်လဲ။)

## 🗣️ Key Sentences (ဥပမာ စာကြောင်းများ - 例文)

1. **ひらがな**: ここは きょうしつです。
   - **Burmese (မြန်မာ)**: ဒီနေရာက စာသင်ခန်း ဖြစ်ပါတယ်။
   - **Kanji**: ここは 教室です。
   - **Romaji**: Koko wa kyōshitsu desu.
   - **English**: This place is a classroom.

2. **ひらがな**: おてあらいは どこですか。
   - **答 (ひらがな)**: あそこです。
   - **Burmese (မြန်မာ)**: သန့်စင်ခန်းက ဘယ်မှာလဲ။ (ဟိုမှာ ပါ。)
   - **Kanji**: お手洗いは どこですか。 (あそこです。)
   - **Romaji**: Otearai wa doko desu ka. (Asoko desu.)
   - **English**: Where is the restroom? (It's over there.)

3. **ひらがな**: この とけいは いくらですか。
   - **答 (ひらがな)**: ごせんえんです。
   - **Burmese (မြန်မာ)**: ဒီနာရီက စျေးဘယ်လောက်လဲ။ (၅၀၀၀ ယန်း ပါ。)
   - **Kanji**: この 時計は いくらですか。 (５,０００円です。)
   - **Romaji**: Kono tokei wa ikura desu ka. (Go-sen en desu.)
   - **English**: How much is this watch? (It is 5,000 yen.)
