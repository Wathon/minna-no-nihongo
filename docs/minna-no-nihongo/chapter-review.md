## 📐 Detailed Breakdown by Chapter

### [[minna-no-nihongo/lesson-01|第1課 Lesson 01 — Self-Introductions (じこしょうかい)]]
- **Key Grammar**:
  - `Noun1 は Noun2 です` — *Noun1 က Noun2 ဖြစ်ပါတယ်။*
  - `Noun1 は Noun2 ではありません` — *Noun1 က Noun2 မဟုတ်ပါဘူး။*
  - `Noun1 は Noun2 ですか` — *Noun1 က Noun2 လား။*
  - `Noun1 の Noun2` — *Noun1 ရဲ့ Noun2*
- **Burmese Particles**: `は` = `က/ဟာ`, `です` = `ဖြစ်ပါတယ်`, `の` = `ရဲ့`.
- **Linked Grammar Notes**: [[grammar/particle-wa]], [[grammar/particle-ka]], [[grammar/particle-no]]
- **Linked Vocab Hubs**: [[vocabulary/occupations]]

### [[minna-no-nihongo/lesson-02|第2課 Lesson 02 — Demonstratives ( Ko-So-A-Do System )]]
- **Key Grammar**:
  - `これ / それ / あれ は Noun です` — *ဒါ / အဲဒါ / ဟိုဒါ က Noun ဖြစ်ပါတယ်။*
  - `この / その / あの Noun` — *ဒီ / အဲဒီ / ဟို Noun*
  - `Noun1 か Noun2 ですか` — *Noun1 လား၊ Noun2 လား။*
  - `Noun の (Pronoun replacement)` — *Noun ရဲ့ ဟာ*
- **Burmese Particles**: `これ` = `ဒါ`, `それ` = `အဲဒါ`, `あれ` = `ဟိုဒါ`.
- **Linked Vocab Hubs**: [[vocabulary/demonstratives]]

### [[minna-no-nihongo/lesson-03|第3課 Lesson 03 — Locations, Directions & Prices]]
- **Key Grammar**:
  - `ここ / そこ / あそこ は Noun です` — *ဒီမှာ / အဲဒီမှာ / ဟိုမှာ က Noun ဖြစ်ပါတယ်။*
  - `Noun は どこ / どちら ですか` — *Noun က ဘယ်မှာလဲ / ဘယ်ဘက်မှာပါလဲ။*
  - `Noun の Noun (Origin/Country)` — *Noun1 နိုင်ငံထုတ် Noun2*
  - `Noun は いくらですか` — *Noun က စျေးဘယ်လောက်လဲ။*
- **Burmese Particles**: `ここ` = `ဒီမှာ`, `どちら` = `ဘယ်ဘက် (ယဉ်ကျေး)`
- **Linked Vocab Hubs**: [[vocabulary/places]]

### [[minna-no-nihongo/lesson-04|第4課 Lesson 04 — Time, Calendar & Verb Forms]]
- **Key Grammar**:
  - `〜時〜分です` — *〜နာရီ 〜မိနစ် ဖြစ်ပါတယ်။*
  - `Verb ます / ません / ました / ませんでした` — *ကြိယာ၏ ယဉ်ကျေးသော ရုပ်ပြောင်းပုံစံများ*
  - `Specific Time に Verb` — *[တိကျသော အချိန်] မှာ ကြိယာ လုပ်တယ်*
  - `Time1 から Time2 まで` — *Time1 မှ Time2 ထိ*
- **Burmese Particles**: `に` = `မှာ`, `から` = `မှ`, `まで` = `ထိ`.
- **Linked Grammar Notes**: [[grammar/verb-masu-form]], [[grammar/particle-ni]], [[grammar/particle-kara-made]]
- **Linked Vocab Hubs**: [[vocabulary/time-calendar]]

### [[minna-no-nihongo/lesson-05|第5課 Lesson 05 — Movement & Transport]]
- **Key Grammar**:
  - `Place へ 行きます / 来ます / 帰ります` — *[နေရာ] သို့ သွားတယ် / လာတယ် / ပြန်တယ်*
  - `Vehicle で 行きます` — *[ယာဉ်] ဖြင့် / နဲ့ သွားတယ်*
  - `Person と 行きます` — *[လူ] နှင့် / နဲ့ သွားတယ်*
  - `いつ 行きますか` — *ဘယ်တော့ သွားမှာလဲ။*
- **Burmese Particles**: `へ` = `သို့ / ကို`, `で` = `ဖြင့် / နဲ့`, `と` = `နှင့် / နဲ့`.
- **Linked Grammar Notes**: [[grammar/particle-he]], [[grammar/particle-de]]
