# Minna no Nihongo Lesson 04 (第4課)

**Status**: 🟩 Mastered
**Date Started**: 2026-07-23
**Focus**: Time (`〜時〜分`), Days of the week (`〜曜日`), Daily Routines, Verb Conjugations (`ます/ません/ました`), Range Particles (`〜から / 〜まで`)

---

## 📌 Overview & Objectives (ရည်ရွယ်ချက်)
- အချိန် (နာရီ၊ မိနစ်) နှင့် ရက်သတ္တပတ် နေ့များကို မေးမြန်း ပြောဆိုနိုင်ရန်။
- နေ့စဉ် လှုပ်ရှားမှု ကြိယာများကို ယဉ်ကျေးသော ပစ္စုပ္ပန်/အတိတ် ပုံစံဖြင့် ပြောဆိုနိုင်ရန် (`〜ます / 〜ました`)。
- အလုပ်/ကျောင်း စတင်ချိန် နှင့် ပြီးဆုံးချိန်ကို `〜から` (မှ) နှင့် `〜まで` (ထိ) သုံး၍ ပြောဆိုနိုင်ရန်။

## 📐 Key Grammar Patterns (သဒ္ဒါ - 文法)

1. **[[grammar/verb-masu-form|Verb ます / ません / ました / ませんでした]]** — *ကြိယာ၏ ယဉ်ကျေးသော ပုံစံများ*
   - Example (ひらがな): べんきょうします (စာလေ့လာပါတယ်)
2. **[[grammar/particle-ni|Specific Time に Verb]]** — *[တိကျသော အချိန်] မှာ ကြိယာ လုပ်တယ်*
   - Example (ひらがな): わたしは ６じはん（６時半）に おきます。 (ကျွန်တော်က ၆ နာရီခွဲမှာ အိပ်ရာထပါတယ်။)
3. **[[grammar/particle-kara-made|Noun1 から Noun2 まで]]** — *Noun1 မှ Noun2 ထိ*
   - Example (ひらがな): ぎんこうは ９じから ３じまでです。 (ဘဏ်က ၉ နာရီမှ ၃ နာရီထိ ဖြစ်ပါတယ်။)
